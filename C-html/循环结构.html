循环体：反复执行的程序段。

5.2 while循环

while(循环条件或表达式)
{
(循环体);
}
  1. 计算表达式的值
  1. 当表达式的值为真时，执行循环体语句。
  1. 再判断表达式的值，重复上述步骤，直到表达式的为假时，结束循环。
  1. 继续执行循环的后续语句。

int a=0;//1. 定义变量a,并将其初始化为0.
while (a<100)//2. 1<100成立,条件成立,该a<100的计算值为真,执行循环体。
{
    ......;
    ......;
    a++;//3 完成a的自加后返回到判断循环条件,初始化循环条件中的a为2，重复循环过程,直到a=100,不满足判断条件,不再执行循环语句。
}

int main ()

5.3 do-while循环

do
{
    (循环体语句);
}
while(表达式);

步骤

  1. 先执行循环体语句
  1. 判断表达式
  1. 当表达式的值为真时,返回重新执行循环体语句,如此反复,直到表达式的值为假时结束循环。

输入一个正整数,将其逆序输出。例如输入12345，输出54321

#include <stdio.h>
int main()
{
    unsigned int a, b;
    scanf("%u", &a);
    do
    {
        b = a % 10;
        a = a / 10;
        printf("%u", b);
    } 
    while (a > 0);
    printf("\n");
}

第一次循环
12345%10=5
12345/10=1234
判断表达式为真

第二次循环
1234%10=4
1234/10=123
判断表达式为真

******
123%10=3
123/10=12
判断表达式为真

******
12%10=2
12/10=1
判断表达式为真

******
1%10=1
1/10=0
判断表达式为假，结束循环体。

5.4 while与do-while的比较

  • while： 先判断后执行，可能一次也不执行。
  • do-while：先执行后判断，至少执行一循环。
  • 二者都是直到表达式为假结束循环
#include<stdio.h>
int main()
{
    int sum = 0, i;
    scanf("%d", &i);
    while(i<=10)
    {
        sum = sum + i;
        i++;
    }
    printf("sum=%d\n", sum);
}

#include<stdio.h>
int main()
{
    int sum = 0, i;
    scanf("%d", &i);
    do
    {
        sum = sum + i;
    } while (i<=10);
    
    printf("sum=%d\n", sum);
}

5.5 for循环

for (表达式1;表达式2;表达式3)
{
    (循环语句);
}

工作原理

  1. 计算表达式1的值
  1. 计算表达式2的值（判断循环条件是否为真）
  1. 若表达式2的真值为真，则执行循环体语句一次。
  1. 计算表达式3的值
  1. 重复步骤2，直到真值为假，循环终止

输入的值为5,求5的阶乘。
#include <stdio.h>
int main()
{
    int number,count,factorial=1;
    printf("\n 请输入任意一个正整数：");
    scanf("%d",&number);
    for(count = 1; count <=number; count++)
		factorial=factorial*count;
    printf("\n %d的阶乘 = %d\n",number,factorial);
}

3应用说明

  1. 省略表达式1即缺少变量赋值，此时应在for语句之前初始化循环变量。
  1. 省略表达式2即缺少判断条件，应在循环体内设置结束循环，否则将因无条件而导致死循环。
  1. 省略表达式3即缺少循环变量的修改，应在循环体内修改循环变量。

5.6 循环嵌套

循环嵌套：一个循环体内又包含另一个完整的循环结构。

#include<stdio.h>
int main()
{
    int i, j;
    for (i = 1; i <= 9; i++)
    {
        for (j = 1; j <= i;j++)
            printf("%d*%d=%d\t", j, i, j * i);
    printf("\n");
    }
    getchar();
    return 0;
}

5.7 转移语句

转移语句：改变程序的正常流向。

1.break语句

break语句用于switch语句和循环语句，作用是跳出本层switch语句或跳出本层循环，转去执行后面的程序。通常与条件语句一起使用。

#include<stdio.h>
int main()
{
    int count = 0, ch;
    printf("\n请输入一个字符:");
    while((ch=getchar()!='\n'))
    {
        if(ch==' ')
            break;
        count++;
    }
    printf("\n共有%d个有效字符。\n", count);
}



//让计数器为count
//输出 请输入一个字符:
//当去最前边字符且不为空格时，执行循环体
//判断ch是否空格，并向下运行
//若ch为空格则跳出循环体，执行输出 共有几个有效字符
//部位空格时，计数器自加，并返回循环条件继续执行
//输出

2.contine语句

只能用于循环，作用是跳过循环体中在、剩余的语句而执行下一次循环。

#include<stdio.h>
int main()
{
    int i,sum;
    for (i = 1; i <= 100;i++)
    {
        if(i%10==3)
        {
            continue;
        }
        sum += i;
    }
    printf("sum=%d\n", sum);
}
