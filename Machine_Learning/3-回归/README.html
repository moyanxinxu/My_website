应用举例

  1. 股市预测
    • 输入:过去十年过票的变动,新闻咨询,公司并购咨询等
    • 输出:预测故事明天的平均值
  2. 自动驾驶
    • 输入:无人车上各个传感器的数据
    • 输出:方向盘的角度
  3. 商品推荐
    • 输入:商品 A 的特性,商品 B 的特性
    • 输出:购买商品 B 的可能性
  4. 宝可梦精灵攻击力预测
    • 输入:进化前的攻击力,物种,血量,重量,高度.
    • 输出:进化后的攻击力

模型步骤

  1. 模型假设 : 选择模型框架(线性模型)
  2. 模型评估 : 判断众多模型的好坏(损失函数)
  3. 模型优化 : 筛选最优的模型(梯度下降)

以一个特征 xcpx_{cp}xcp​ 为例子,线性模型假设 y=b+w∗xcpy = b + w * x_{cp}y=b+w∗xcp​ ,所以 www 和 bbb 可以猜测很多模型 :

f1:y=10.0+9.0×xcpf_1: y = 10.0 + 9.0 \times x_{cp}f1​:y=10.0+9.0×xcp​
f2:y=9.8+9.2×xcpf_2: y = 9.8 + 9.2 \times x_{cp}f2​:y=9.8+9.2×xcp​
f3:y=−0.8−1.2×xcpf_3: y = -0.8 -1.2 \times x_{cp}f3​:y=−0.8−1.2×xcp​

虽然可以做出很多假设,但是对于f3:y=−0.8−1.2×xcpf_3: y = -0.8 -1.2 \times x_{cp}f3​:y=−0.8−1.2×xcp​的假设是不合理的!

多元线性模型(多个特征)

  • wiw_iwi​ : 各种特征的权重

如何判断众多函数的好坏呢

从 数学 的角度来讲,我们使用距离.求出进化前的攻击力和模型预测的攻击力的差值,来判定模型的好坏.也就是说我们使用了损失函数来衡量模型的好坏,统计 10 组原数据(y^n−f(xcpn))2(\widehat{y}^n - f({x_{cp}^n}))^2(y​n−f(xcpn​))2的和,和越小,则模型越好.

公式推导过程 :

L(f)=∑n=110(y^n−f(xcpn))2=∑n=110(y^n−(b+w×xcp))2\begin{align} L(f) &= \sum_{n=1}^{10}(\widehat{y}^n - f(x_{cp}^n))^2\\ &= \sum_{n=1}^{10}(\widehat{y}^n - (b + w \times x_{cp}))^2\\ \end{align}L(f)​=n=1∑10​(y​n−f(xcpn​))2=n=1∑10​(y​n−(b+w×xcp​))2​​

最终定义损失函数:
L(w,b)=∑n=110(y^n−(b+w×xcp))2L(w,b) =\sum_{n=1}^{10}(\widehat{y}^n - (b + w \times x_{cp}))^2L(w,b)=n=1∑10​(y​n−(b+w×xcp​))2
